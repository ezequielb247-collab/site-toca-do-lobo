# Site profissional — Toca do Lobo Jiu-Jitsu

Versão atualizada do site da Toca do Lobo Jiu-Jitsu — GFTeam Aroeira.

## Publicação no GitHub

1. Extraia o ZIP.
2. No repositório `site-toca-do-lobo`, substitua os arquivos existentes.
3. Envie também a pasta `assets`.
4. Confirme o commit.

O Render fará o novo deploy automaticamente, mantendo a mesma URL.

## Conteúdo incluído

- Logo da academia
- Imagem de capa com treino
- Seção da estrutura
- Seção do professor Junior Lobo
- Horários
- Mensalidades
- Galeria e conquistas
- WhatsApp integrado
- Google Maps integrado
- Layout responsivo

## Produção

- URL: https://site-toca-do-lobo.onrender.com
- Hospedagem: Render Static Site
- Branch de produção: `main`
- Deploy automático a cada commit

O site inclui SEO, dados estruturados, galeria acessível, integração com WhatsApp, Instagram e Google Maps.
